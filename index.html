Ottawa

Skateboard Parks

Design features

  • Home screen will drop pins on the places where the skate board parks are located.
  • Hover over any of the pins will show a brief description of the place along with address and/or an image.
  • A pointer icon on the screen will allow user to search for places relative to the current location of the user.
  • Input field on the main screen will allow the user to enter the location close to which user want to find the parks.
  • Three navigations will allow user to sort the list according to how cool the places are, how clean they are and how friendly people are.
  • Click on any of the pins will take you to another page with detail information distance from current location and means of transportation (car, bus, walk) user can rate the place and leave comments also.
